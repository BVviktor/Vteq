Bolt force and tightening torque calculator tool

Tools used: C# (WindowsForms), XML (input/output)

Cranktrain force/torque calculator app with balancer optimization tool

Tools used: C# (WindowsForms), XML (input/output), Matlab, Excel, Photoshop
OS: Windows, PC
Description:This is a simple tool that calculates the cranktrain force/torque based on the input parameters. It also generates an XML file with the results. It is capable for balancer shaft optimizations to minimalise the torques and to reset the crankshaft forces to zero.

Testbed manager tool for clutch coupling test

Tools used: NodeRed, Python, Grafana, InfluxDB, Excel
OS: Linux, Raspberry PI
Description:This is a simple user interface (UI) for a testbed. The testbed is capable of testing the clutch couplings in different conditions (temperature, pressure, etc.). It is capable of controlling the testbed, collecting data, and generating reports. It also generates an XML file with the results which are afterward transfered from Raspberry to PC wireless.
